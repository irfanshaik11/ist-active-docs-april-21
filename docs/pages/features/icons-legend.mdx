# UI Icons & Badges Legend

Interstate packs a lot of information into tight UI space via icons and colored badges. This page is the reference for what every one of them means.

---

## Pulse Column Headers

| Label | Meaning |
|---|---|
| **New Pairs** | Tokens just launched on a tracked launchpad (0-80% bonding) |
| **Final Stretch** | Tokens near migration (80-99% bonding); high-volume window |
| **Migrated** | Tokens successfully moved to a DEX pool |

---

## Detection Flags (per token row)

See [Bundle & Sniper Detection](/analytics/bundle-detection) for how each is computed.

| Icon | Flag | Meaning |
|---|---|---|
| 🤖 | **Sniper** | Wallet bought within slots 0-3 of creation (~1.6s). Bot-like behavior. |
| 🔐 | **Insider** | Wallet bought in the exact same slot as the creator's first buy. Team/coordinated. |
| 🤝 | **Bundler** | Wallet was part of a 4+ wallet bundle in the same slot. Coordinated manipulation. |
| 👨‍💻 | **Dev** | Creator's wallet. Tracks their current holding live. |
| 🎯 | **Top 10** | Combined balance of the 10 biggest holders. |
| 📋 | **Copy Trader** | Wallet detected as following another wallet's trades. |

**Percentage reading:**
- Each flag's `%` = supply held by flagged wallets ÷ total supply
- Higher = more concentration in that category = more risk

## Safety Indicators (CA Security Check)

See [Security Check](/features/security-check) for details.

| Badge | Meaning | Action |
|---|---|---|
| 🟢 **Green dot** | All critical checks passed | Relatively safe on contract level |
| 🟡 **Yellow dot** | Soft flags present (concentration, past rugs) | Investigate before sizing up |
| 🔴 **Red dot** | Critical flag (active mint, freeze, LP in dev wallet) | Likely unsafe — don't trade |

### Specific Contract Flags

| Icon | Meaning |
|---|---|
| 🔥 | **LP burned** — liquidity locked forever, dev can't rug |
| 🔒 | **LP locked** — liquidity in a time-lock contract |
| ⚠️ **LP in dev** | **LP in dev wallet** — dev can pull liquidity any moment |
| ✅ **Mint renounced** | Creator cannot print more tokens |
| ❌ **Mint active** | Creator can mint unlimited supply — massive dump risk |
| ✅ **Freeze renounced** | Creator cannot freeze wallets |
| ❌ **Freeze active** | Creator can freeze your wallet, preventing sells |

## Glossary: Color Meanings

| Color | Used for |
|---|---|
| **Indigo** (`#6366f1`) | Interstate brand, primary actions, active nav |
| **Green** (`#10b981`) | Positive (gains, safe flags, buys, live status) |
| **Red** (`#ef4444`) | Negative (losses, risk flags, sells, failures) |
| **Yellow** (`#f59e0b`) | Warning / ambiguous (soft flags, flagged wallets) |
| **Pink / Magenta** (`#ec4899`) | Arena rewards, cashback, milestones |
| **Gray** (`#6b7280`) | Secondary text, disabled states, dust positions |
